\Ldap\Schema

\Ldap\SchemaAbstractSchemaElement

Abstract class with code common to all schema elements.

This comes down (as per RFC4512) to the OID and a description. Yet this class also provides means to check for the most common ABNF productions defined in RFC4512, such as keystring, oid, kind, etc.
